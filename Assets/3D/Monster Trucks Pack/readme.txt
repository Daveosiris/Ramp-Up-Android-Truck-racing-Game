The cars are split in hierarchy, so that the body can be moved without affecting the tires. 
Wheels and tires are attached to spindles, which can rotate while following the axles.
Suspension is intended to use simple rotating movement.



Use "base color" layers to adjust color in Albedo map, paint as you wish. 
"livery" folder includes overlay graphics, you can hide them or replace as you like.
The wheels have a similar color layer you can use to paint them.
